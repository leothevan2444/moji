# 自动化

自动化设置控制后台轮询、StashBox 优先级、订阅决策和自动选种排序。修改会影响之后的操作。

## 调度与发行策略

任务进度间隔控制 qBittorrent 同步，订阅间隔控制演员定时检查。填写 0 时分别采用内置默认值 60 秒和 1 小时；负数会关闭对应循环。

发行策略分别控制独演、共演和总集类影片。**下载** 创建任务，**复核** 只记录发行内容，**阻止** 不下载。总集信号优先于演员人数分类。发行日期缺失或超出自动下载范围时，原本的自动下载会转为复核。

## 数据源与选种规则

StashBox 端点从 Stash 读取。发现搜索在第一个有结果的端点停止；订阅查询按保存顺序尝试。未列入自定义顺序的端点仍按 Stash 原顺序追加。

选种采用顺序规则链：第一条能区分候选的规则决定顺序，后续规则只打破平局。快速规则使用索引器、标题、日期、相似度、做种数或大小；文件规则只检查前 N 个候选，可偏好单视频或内部文件名。`PREFER` 提前匹配项，`AVOID` 后移匹配项，文件名 `LOCK` 提供最强偏好。无效正则不会匹配。
