# 开始使用

Moji 是 Stash 的伴随服务：发现发行内容、把下载提交给 qBittorrent、将已完成内容交付到 Stash 媒体库，并通知 Stash 扫描。媒体的编目和展示仍由 Stash 负责。

> Moji 不会替你安装或配置 Stash、Jackett、qBittorrent 或 StashBox。

## 使用前准备

你需要正在运行的 Stash、Jackett 和 qBittorrent。Jackett 中至少要有一个已配置索引器。若要使用元数据发现和演员订阅，还需在 Stash 中配置至少一个 StashBox 端点；端点要求鉴权时必须填写 API key。

Moji 服务端必须能访问这三个服务。使用文件交付时，Moji 还必须能读取下载目录并写入媒体库目录。

## 首次配置

1. 打开 **设置 > 连接**，分别保存三个服务的 URL 和凭据。
2. 返回 **主页**。只有最近一次探测成功才表示服务“就绪”；“已配置”不等于连接正常。
3. 打开 **设置 > 入库**。qBittorrent 与 Stash 已能访问同一份文件时选择路径映射；需要 Moji 复制、移动或链接文件时选择文件交付。
4. 打开 **设置 > 自动化**。设置进度和订阅间隔，刷新 StashBox 列表，并检查发行策略与自动选种策略。
5. 在 **发现** 中搜索，或从 **演员** 详情加入影片。随后在 **任务** 中跟踪，直到 Stash 扫描完成。

## 安全完成首次任务

先选一个结果唯一的番号。确认路径前，文件交付动作保持为 **复制**。检查 qBittorrent 内容路径确实位于 qB 下载根目录下，最终扫描路径确实位于 Stash 媒体库内。任务完成后先在 Stash 中验证影片，再扩大订阅自动化范围。
