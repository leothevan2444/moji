# 产品简介

Moji 是 Stash 的自托管伴随服务，负责协调发行内容发现、种子选择、qBittorrent 下载、媒体库交付和最终的 Stash 扫描。

## Moji 管理的内容

Moji 保存自己的下载任务和演员订阅；通过 Stash 中配置的 StashBox 端点搜索元数据，通过 Jackett 搜索种子。下载完成后，Moji 会先将内容映射或交付到 Stash 媒体库，再请求 Stash 扫描。

Stash 仍是演员、媒体库影片、媒体路径和内容展示的权威来源；qBittorrent 是种子传输状态的权威来源；Jackett 负责索引器配置和搜索可用性。

## 主要流程

1. 在 **发现** 或演员详情中找到影片。
2. 从结构化元数据或指定种子创建任务。
3. 由 Moji 选种并跟踪 qBittorrent 下载。
4. 将完成的内容映射或交付到媒体库。
5. 由 Stash 扫描计算出的路径。

只有关联的 Stash 扫描任务完成后，Moji 才会把任务标记为完成。
