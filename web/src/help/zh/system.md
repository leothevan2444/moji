# 系统

系统设置控制任务删除的联动范围和 Moji 的上游图片缓存。删除任务前应确认当前删除策略。

## 任务删除

策略可以只删除 Moji 任务记录、同时删除对应 qBittorrent 任务，或连同下载文件一起删除。策略在实际删除时读取，不追溯既有操作；删除确认窗口会显示当前效果。

## 图片缓存与诊断

上游图片始终经 Moji 代理。关闭磁盘缓存后不再写入新缓存，但仍可读取已有文件。容量范围为 64–20480 MB，保留时间为 1–365 天。清理会删除过期项并缩减超限缓存。清空缓存只删除本地图片，保留来源登记，因此之后仍可重新获取。

日志支持按级别筛选、刷新、复制和下载。应先检查服务状态与任务错误，再使用日志定位后端原因。关于页显示当前 Moji 版本。
